# MeruUniversity-SMS

A modern, modular School Management System (SMS) for Meru University.

## Features
- **Student Management:** Add, view, and manage student records.
- **Course Management:** Register courses, allocate students, track enrollments.
- **Fee Tracking:** Record payments, view fee status.
- **Library Management:** Add books, borrow/return, track availability.
- **Performance Analytics:** Record scores, view top performers.

## Technologies
- **Backend:** Python, Flask, SQLite
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Deployment:** Render

## Installation
1. Clone the repo: `git clone https://github.com/YourUsername/MeruUniversity-SMS.git`
2. Navigate to backend: `cd backend`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python app.py`
5. Open in browser: `http://127.0.0.1:5000`
